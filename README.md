# CS + Fitness Portfolio

A simple personal portfolio website built with semantic HTML and clean CSS.
It highlights my background as a Computer Science student and my structured approach to fitness.

## Tech Used

- HTML5 (semantic structure)
- CSS3 (typography, spacing, layout)
- Git/GitHub

## Features

- Single-page layout with section navigation
- Accessible navigation (ARIA label)
- Clean typography and readable layout
- Simple, professional color palette (grays + blue)

## Sections

- About
- Skills
- Projects
- Training
- Contact

## What I Learned

- How to structure a page using semantic HTML (`header`, `nav`, `main`, `section`, `footer`)
- How to use IDs and anchor links for navigation
- How to style a page with basic CSS (spacing, typography, layout)
- How to improve accessibility with small best practices

## How to Run Locally

1. Download or clone the repository
2. Open `index.html` in your browser
